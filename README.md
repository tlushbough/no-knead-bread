# No-Knead Bread Website

A beautiful, responsive website showcasing the art of no-knead bread baking. This is a learning project to practice HTML, CSS, and JavaScript.

## Project Structure

```
no-knead-bread-site/
├── index.html          # Main homepage
├── css/
│   └── style.css      # All styling
├── js/
│   └── script.js      # Interactive features
├── images/            # Put your bread photos here
├── recipes/           # Individual recipe pages
│   └── classic.html   # Classic no-knead bread recipe
└── README.md          # This file
```

## How to View the Website

1. **Open in a Browser:**
   - Navigate to the project folder
   - Double-click `index.html` to open it in your default web browser

2. **Using a Local Server (Recommended):**
   - If you have Python installed, open Terminal and navigate to the project folder
   - Run: `python3 -m http.server 8000`
   - Open your browser to: `http://localhost:8000`

## Features

- **Responsive Design:** Works on desktop, tablet, and mobile devices
- **Smooth Scrolling:** Navigation links smoothly scroll to sections
- **Animated Elements:** Content fades in as you scroll
- **Recipe Pages:** Detailed recipe instructions with ingredients and tips
- **Modern Layout:** Clean, bakery-inspired design with warm colors

   - CSS variables for theming

3. **JavaScript:**
